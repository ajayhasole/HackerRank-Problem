# Counting Valleys

## Input

The function accepts two parameters:
1. `steps` (INTEGER): The number of steps in the hike.
2. `path` (STRING): A string where each character represents a step in the hike:
   - `U` for an uphill step
   - `D` for a downhill step

### Example Input
```
8
UDDDUDUU
```

## Output

The function returns an INTEGER representing the number of valleys traversed during the hike.

### Example Output
```
1
```

## Explanation

In the example provided:
- The path `UDDDUDUU` represents a sequence of steps that goes down into one valley and comes back to sea level once.
